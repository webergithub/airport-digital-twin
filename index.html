机场数字孪生 · AirportTwin

机场数字孪生 · 数字机场仿真平台

AirportTwin 是一个开源的机场数字孪生（数字机场 / Airport Digital Twin）仿真平台，在浏览器中以三维方式还原机场地面运行全流程，无需安装任何软件。

  • 真实比例的波音 / 空客机型，含起飞、降落、爬升、下滑真实剖面
  • 廊桥自动对接与旅客下机动画
  • 13 节点航班保障链路：清洁、加油、配餐、行李装卸、饮水、污水等
  • 跑道单机排队起降与 DMAN 离场排序（TSAT 机位等待）
  • A-CDM 里程碑全记录，准点率与燃油节省 KPI
  • 三层架构：仿真数据层 + 分析优化算法层 + 三维孪生视图层
  • 标准 JSON 数据接口，可接入真实机场运行数据，适用于机场孪生技术演示、教学与调度算法验证

About (English)

AirportTwin is an open-source airport digital twin that simulates complete airport ground operations in your browser: Boeing/Airbus aircraft, jet-bridge docking, a 13-node turnaround workflow, runway sequencing, DMAN departure metering and A-CDM milestones — with a standard JSON API for connecting real airport data.
